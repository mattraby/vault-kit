# Knowledge Vault (OKF / LLM-wiki starter)

A starter knowledge vault: an Obsidian-compatible, portable **Open Knowledge Format (OKF) v0.1**
bundle maintained in the **LLM-wiki** style. This directory is a generated export of
[vault-kit](https://github.com/mattraby/vault-kit) for people who want the template without
installing the Claude Code plugin.

## Use it

1. Copy this directory to your project (e.g. `degit mattraby/vault-kit/template my-project`).
2. Open `vault/` in [Obsidian](https://obsidian.md).
3. Read `vault/AGENTS.md` — the schema and workflows. Codex and other AGENTS.md-aware tools
   read it natively; Claude Code loads it through the one-line `vault/CLAUDE.md` bridge.
4. Rename the placeholder wiki categories; delete the two worked examples once the pattern is clear.
5. Record durable project facts in `vault/MEMORY.md` — it travels with the repo.
6. Helper scripts (extraction, conformance check, lint scan) live in `vault/.bin/` for any
   agent; the `ingest-source` and `lint-vault` skills in `.claude/skills/` wrap them for
   Claude Code.
